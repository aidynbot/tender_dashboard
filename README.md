# 🏗️ Тендерная аналитика Pro

Интерактивный дашборд для анализа строительных тендеров — построен на **Streamlit** и **Plotly**. Позволяет отслеживать выигранные тендеры, анализировать маржинальность, визуализировать географию и моделировать сценарии «что если?».

---

## 📸 Возможности

| Блок | Описание |
|---|---|
| 🎯 Скоринг (AI Light) | Автоматический расчёт балла перспективности (0–100) для каждого тендера |
| 🔮 What-if симуляция | Прогноз выручки и прибыли при изменении маржи и конверсии |
| 💹 Price Intelligence | Анализ снижения от НМЦК, ценовые инсайты по заказчикам |
| 📈 Динамика | Выручка и прибыль с группировкой по месяцу / кварталу / году |
| 📅 LFL-сравнение | Сравнение двух выбранных лет Like-for-Like |
| 💰 Финансы | Маржинальность по типам работ, структура себестоимости |
| 🗺️ Карта | Геокарта тендеров по регионам Казахстана |
| 🏢 Заказчики | Топ заказчиков, Win Rate, объём vs количество |
| ⚔️ Конкуренты | Win/Loss по конкурентам |
| 📋 Таблица | Умный поиск, сортировка, цветовая индикация дедлайнов |
| 📄 PDF-отчёт | Генерация готового отчёта для руководителя |

## 🚀 Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
cd tender_dashboard
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустить дашборд

```bash
streamlit run tender.py
```

Дашборд откроется в браузере по адресу `http://localhost:8501`.

## 📥 Загрузка своих данных

В сайдбаре нажмите **«📂 Загрузить Excel»** и выберите `.xlsx`-файл.

Дашборд автоматически распознаёт столбцы по ключевым словам. Рекомендуемые названия столбцов:

| Столбец | Синонимы |
|---|---|
| Дата подачи | date, дата |
| Заказчик | customer |
| Тип работ | тип, вид, type, work |
| Регион | region |
| Сумма тендера | сумма, цена, amount, price, sum |
| НМЦК | нмцк, nmck, начальн |
| Себестоимость | себест, cost |
| Статус | status |
| Конкурент | конкур, competitor |
| Валюта | валют, currency |

Если нужных столбцов нет — они будут рассчитаны автоматически (например, себестоимость = 82% от суммы).

Шаблон Excel можно скачать прямо из сайдбара кнопкой **«⬇️ Шаблон Excel»**.

## 🎯 Алгоритм скоринга

Балл перспективности (0–100) рассчитывается по 5 факторам:

- **+20** — исторический Win Rate у данного заказчика
- **+15** — высокая маржа по типу работ (>22% → +15, >17% → +8)
- **+10** — регион входит в топ-3 по конверсии
- **+10** — сумма тендера выше 75-го перцентиля
- **−10** — просроченный дедлайн

| Балл | Рекомендация |
|---|---|
| ≥ 65 | ✅ Рекомендуем |
| 45–64 | ⚠️ Средний риск |
| < 45 | ❌ Высокий риск |

## 💱 Мультивалютность

Дашборд поддерживает отображение сумм в **KZT, RUB, USD, EUR**. Курсы загружаются автоматически с открытого API (`open.er-api.com`). При отсутствии интернета используются встроенные дефолтные значения. Курс можно задать вручную.

## 🗺️ Геокарта

Координаты определяются по названию региона через встроенный словарь (~50 городов и областей Казахстана). Если регион не распознан — точка размещается в центре Казахстана (48.02°N, 66.92°E). Нераспознанные регионы выводятся в сайдбаре в блоке **«⚠️ Регионы без точных координат»**.

## 📄 PDF-отчёт

Нажмите **«📄 Сформировать отчёт для директора»** на главной странице. Отчёт включает:

- KPI-плашки (выручка, прибыль, маржа, конверсия)
- Маржинальность по типам работ
- Ближайшие дедлайны
- Скоринг тендеров «В процессе»
- Топ заказчиков
- Автоматические инсайты

Для корректного отображения кириллицы требуется наличие TTF-шрифта с кириллицей в системе (`DejaVuSans`, `LiberationSans`, `FreeSans` или `Arial`). При их отсутствии используется встроенный Helvetica.
